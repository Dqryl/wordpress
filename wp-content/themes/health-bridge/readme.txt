== Description ==

Health Bridge is a purpose-built WordPress theme crafted for health and medical websites. With its sleek design and user-friendly interface, this theme offers seamless navigation and essential features tailored to meet the specific needs of healthcare professionals, making it an ideal choice for showcasing medical services and information effortlessly.

== Changelog ==
1.0.0 Initial release

= 1.0.2 =

== Installation ==

1. Go to Themes > Add New, search for "Health Bridge" and then click "Install".
2. After installaion is done click on "Activate".
4. Setup Site Logo and Navigation Menu.
5. Enjoy!
